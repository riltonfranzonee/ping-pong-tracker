## :information_source: How to run

```bash
# Clone the repository
$ git clone https://github.com/riltonfranzonee/ping-pong-tracker

# Go into the directory
$ cd ping-pong-tracker

# Install the dependencies
$ yarn (OR npm install)

# Initialize the app
$ react-native run-ios
```
